# housework-mini

家政人员月薪计算微信小程序，纯前端实现，无后端依赖。

## 功能

- 设置日薪
- 按月份设置应上天数和满勤工资
- 按月份查看日历
- 点击日期后设置为上班、带薪或不上班
- 上班支持全天或半天，半天按 0.5 倍日薪计算
- 带薪状态支持输入倍数，比如 2 薪、3 薪
- 上班和带薪互斥
- 底部实时计算本月工资
- 数据保存到微信小程序本地缓存

## 本地开发

1. 打开微信开发者工具
2. 导入当前目录
3. AppID 可先选择测试号或游客模式
4. 编译运行

## 数据存储

当前版本使用 `wx.setStorageSync` / `wx.getStorageSync` 保存数据，key 为：

```text
housework-salary-state-v1
```

数据结构大致如下：

```json
{
  "dailySalary": "300",
  "monthRules": {
    "2026-05": {
      "expectedWorkDays": "26",
      "fullMonthSalary": "7500"
    }
  },
  "months": {
    "2026-05": {
      "2026-05-01": {
        "type": "work",
        "multiplier": "1"
      },
      "2026-05-02": {
        "type": "paid",
        "multiplier": "2"
      }
    }
  }
}
```

纯本地存储适合单机使用。如果后续需要换手机同步、多人共享或备份恢复，可以再加微信云开发或导入导出功能。
